##  Project Overview

This repository contains the deliverables for **Cyber Security Task 2** assigned by Future Interns.

The objective of this task was to simulate the role of a **Security Analyst (SOC / Blue Team)** by:

- Analyzing phishing email samples
- Identifying phishing indicators
- Performing email header analysis
- Investigating malicious links
- Classifying email risk levels
- Creating a professional awareness report for end users

This project focuses on **cybersecurity awareness and threat analysis**, not exploitation or hacking.

---

##  Objectives

The main goals of this task were:

- Identify common phishing techniques used in email attacks
- Analyze email headers for spoofing and authentication failures
- Detect malicious or suspicious URLs
- Classify emails based on risk level (Safe / Suspicious / Phishing)
- Produce a clear, non-technical awareness report for users and organizations
- Demonstrate practical SOC-level analytical thinking
